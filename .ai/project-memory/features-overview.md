# Features Overview

- **Dashboard**: overview stats and recent sessions
- **Athlete tracking**: profiles, sessions, exercise logs, progress metrics
- **Exercise library**: library with movement pattern tags
- **Prescriptions**: training plans assigned to specific athletes
- **Recovery protocols**: evidence-based recovery protocol browser
- **Student comparison**: side-by-side athlete data comparison
- **Oura integration**: sleep, HRV, activity data from Oura ring
- **Admin users**: role management for admin/moderator/user
- **AI Builder**: conversational interface for admins to request features
  - Classifies intent: conversation, planning, or build
  - Creates GitHub issues automatically when intent = build
  - Triggers the AI engineer pipeline
